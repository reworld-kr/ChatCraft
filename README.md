# ChatCraft

> Windows용 메신저 채팅 스크린샷 생성기 — 당근마켓 / 텔레그램 프리셋 / 라이트 다크 테마 / PNG 내보내기.

[ReWorld](https://reworld.kr) 데스크톱 앱의 **release 배포 채널**입니다. 소스 코드는 별도 모노레포(`reworld-kr/ReWorld`)에서 관리되고, 이 repo는 빌드 산출물(`ChatCraft.exe`, NSIS installer, checksums)만 호스팅합니다.

## 다운로드

[**최신 버전 다운로드 ↓**](https://github.com/reworld-kr/ChatCraft/releases/latest)

- `ChatCraft.exe` — bare 실행 파일 (압축/설치 없이 즉시 실행)
- `ChatCraft_Setup.exe` — NSIS 설치 마법사 (사용자 디렉터리에 설치, UAC 불필요)
- `checksums.txt` — SHA-256 무결성 검증

설치 후 자동 업데이트가 켜져 있어 새 버전 출시 시 다음 실행에 백그라운드 적용됩니다.

## 주요 기능

- 당근마켓 레이아웃 지원 — 텔레그램 등 추가 플랫폼 준비 중
- 메시지 / 이미지 / 날짜 구분선 자유 배치
- 시간 자동 입력으로 자연스러운 대화 흐름
- PNG 한 장으로 내보내기
- 여러 채팅 시나리오 라이브러리 관리
- 라이선스 통합 — 한 계정으로 다른 ReWorld 앱과 함께

## 시스템 요구사항

- Windows 10 / 11 (x64)
- 사용자 디렉터리 설치 → 관리자 권한 / UAC 불필요
- 자동 업데이트 — GitHub Release 채널 모니터링
